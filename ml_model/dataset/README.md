# Dataset Folder

Contains sample CSV datasets for offline ML classifier:

- `heart_attack.csv` – Simulated heart attack emergency data
- `respiratory_emergency.csv` – Simulated respiratory emergency data

Columns:

- symptom_* : 0 (no) / 1 (yes)
- severity : scale 1-5
- age : user age
- emergency : 0 (non-emergency), 1 (emergency)

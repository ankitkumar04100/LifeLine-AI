# Emergency Classification Model

The model is designed to classify emergencies into categories:

- accident
- heart_attack
- bleeding
- burn

It is **lightweight and optimized for offline, on-device inference** using TensorFlow Lite.

Features:
- Fast inference (<1 second)
- Low memory footprint
- Panic-friendly response

Future improvements:
- Include more emergency types
- Multi-language support
- Integration with wearable sensors
